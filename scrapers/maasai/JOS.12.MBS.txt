Ilaiguanak ootimira Musa
1 Kulo apa ilaiguanak le ina kop, ootimira iltung'ana le Israel, neya ina kop enye te lido kekun Yordan te nkalo e nkilepunoto e nkolong ebaiki oerata le Arnon o metabaiki oldoinyo le Hermon, o Araba pooki te nkalo o loosaen: 2Sihon olaiguanani loo Lamori apa omanya Heshbon, naa ninye apa oitore ebaiki Aroer natii olomanaa lo erata le Arnon, o metabaiki empolos o erata o metabaiki olkeju le Yabok, ilo polosie loo Lamori, aa taa, olpisiai le Gilead, 3O Araba o metabaiki enaiposha e Kineret te nkalo o loo saen, o te nkalo e Bet-yeshimot, o metabaiki enaiposha e Araba, ina aiposha naisukut, enkalo e kopikop o metabaiki abori lelo jurumi le Pisga; 4o Oog olaiguanani le Bashan, obo apa lo lpisiai le Refaim, apa omanya Ashtarot o Edrei, 5naa ninye apa oitore oldoinyo le Hermon o Saleka o Bashan pooki o metabaiki olpolosie loo Lkishi o Lmaakati, o lpisiai le Gilead o metabaiki olpolosie le Sihon olaiguanani le Heshbon. 6 Etimira apa ninche Musa osinka lo Laitoriani te nebo iltung'ana le Israel; neishooyo Musa ilo sinka lo Laitoriani enkop enye metaa enkop oo le Ruben, ole Gad olpisiai lo losho le Manase.
Ilaiguanak ootimira Yoshua
7Naa kulo apa ilaiguanak le ina kop ootimira Yoshua o ltung'ana le Israel te nkalo o loontoluo le Yordan, ebaiki Baal-gad ti atua oerata le Lebanon o metabaiki ilo doinyo le Halak, oshumunye aing'or Seir (neisho Yoshua ina kop enye iloshon le Israel meitaa enenye te risioroto enkiririkinoto enye, 8ti atua ina kop nagum, ti atua Araba, o tiatua nkishong'ishong'at, ti atua ong'ata ooti atua negeb, ina kop oo Lhiti, o Lamori, o Lkanaani, o Lperisi, o Lhafi o Lyebusi): 9olaiguanani le Yeriko; obo, olaiguanani le Ai aa ninye natashare Betel, obo: 10olaiguanani le Yerusalem, obo; olaiguanani le Hebron, obo; 11olaiguanani le Yarmut, obo; olaiguanani le Lakish, obo; 12olaiguanani le Eglon, obo; olaiguanani le Geser, obo; 13olaiguanani le Debir, obo; olaiguanani le Geder, obo; 14olaiguanani le Horma, obo; olaiguanani le Aradi, obo; 15olaiguanani le Libna, obo; olaiguanani le Adulam, obo; 16olaiguanani le Makeda, obo; olaiguanani le Betel, obo; 17olaiguanani le Tapua, obo; olaiguanani le Hafer; obo; 18olaiguanani le Afek, obo; olaiguanani le Lasharon, obo; 19olaiguanani le Maden, obo; olaiguanani le Hasor, obo; 20olaiguanani le Shimron-meron, obo; olaiguanani le Akshaf, obo; 21olaiguanani le Taanak, obo; olaiguanani le Megido, obo; 22olaiguanani le Kedesh, obo; olaiguanani le Yokneam ti atua Karmel, obo; 23olaiguanani le Dori te Nafat-dor, obo; olaiguanani le Goiim ti atua Galilaya, obo; 24olaiguanani le Tirsar, obo: ore pooki neaku tomoni uni o obo.
