Ilomon ooinosua Yohana Olaibatisani
1Ore to lari le tomon o miet le nkitoria o laiguanani kitok oji Tiberia Kaisaria, aa Pontio Pilato olbalusui le Yudea, naa Herode olkitok le Galilaya, naa Filipo olalashe lenye olkitok le Iturea o Trikoniti, naa Lisenias olkitok le Abilene, 2naa Annas o Kaiafas ilapolosak kituaak, nebaiki ororei le Nkai Yohana olayioni le Sakaria ti ong'ata. 3Nelo ina kop pooki oo mbat o lkeju le Yordan elikioo enkibatisa naitoduaaya enkirridunoto nayau e mpalakinoto oo ng'ok, 4 anaa enasira te mbuku e Isaya oloiboni le Nkai ajo,
“Oltoilo lo lobuakita ti ong'ata,
entobiraki Olaitoriani enkoitoi,
entorioi irrekiei lenyena
5nenuki nong'ot pooki,
ore pooki doinyo o lkiushin neitodorropi;
ore ineekordilo nererei,
ore nkoitoi naasur neisiashayieki;
6pee edol iltung'ana pooki enjeunoto e Nkai.”
7 Nejoki Yohana lelo ororani ooetuo enetii ninye meibatisa, “Na ishoi oo lasuriaa! Ae ng'ae naikoo intae eng'atai engoro nalotu? 8 Entadedeyie naa enkirridunoto inyi too lng'anayio lenyena; nimintokiki ake ajo, ‘Kiata Abraham aa papa lang.’ Ajoki intae, keidim Enkai ainyeyieki Abraham nkera te kulo soito. 9 Eidipaki taata aaiteleiki entolu ntonat oo lkeek; ore olchani ake pooki lemeiu lng'anayio sidain nedung'i aauroo nenang'akini enkima.”
10Neikilikuan lelo ororani ninye aajoki, “Kaa naa kiaas?” 11Newaliki ajoki, “Ore oloata nkilani are, inchoo eisho nabo ilo lemeeta, ore oloata endaa neng'arie olemeeta.” 12 Nepuonu sii ninche iloosotu kodi meibatisai, neikilikuan aajoki, “Lo aiteng'enani, kaa kiaas iyiook?” 13Nejoki ninche, “Emintulus aaoru ltung'ana enalang ina naitanapaki intae.” 14Neikilikuan sii ninche sikarini aajoki, “Oo iyiook, kaa kiaas?” Nejoki ninche, “Emisim oltung'ani anaa nteenikiki elejare, ninchosho sii ebaiki intae olekoisiayio linyi.”
15Ore elimana ltung'ana eanyita enaaku, neitiko ltauja le pooki te nkaraki Yohana, emusumusu aajo ebaiki naa Kristo ninye, ana mme ninye, 16newaliki Yohana ninche pooki ajoki, “Esipa aibatisa nanu intae te nkare, kake kelotu ologol te nanu lemeishiaakino nalak enkeene e namuke enye. Ilo oibatisa intae te Nkiyang'et Sinyati o te nkima; 17neibung'ita ninye esilet enye te nkaina pee eor asip katukul olkurroto lenye oidong'ishoreki, nesotoki enkano empeche; kake ore suut, nepej te nkima nemearayu ntarasi.”
18Ore te kulie kumok nereiki aikok olorere eliki lomon supati. 19 Kake ore Herode ilo balusui pee eisikong ninye te nkaraki nayama Herodias enkitok o lalashe lenye, o sii te nkaraki kulie bususi naataasa Herode, 20neponiki ena nena pooki neikenoo Yohana te korokoroni.
Eibatisaki Yesu
(Mat 3.3-17; Mk 1.9-11)
21Ore eibatisaki iltung'ana pooki neidipaki aaibatisa sii ninye Yesu, eomonito Enkai nebolo shumata, 22 nedou Enkiyang'et Sinyati etiu anaa enturkulu adoiki ninye. Nelotu oltoilo oing'uaa shumata ojo, “Ira iyie Oinoti lai lanyor, kaashipakino iyie.”
Entipat e Yesu
(Mat 1.1-17)
23Ore pee eiteru Yesu enkisiaayiare enye naa oloo larin oobaya tomoni uni. Ore ninye anaa enajo iltung'ana, olayioni le Daudi, olayioni le Heli, 24olayioni le Matat, olayioni le Lawi, olayioni le Matat, olayioni le Lawi, olayioni le Melki, olayioni le Yanai, olayioni le Yosef, 25olayioni le Matatias, olayioni le Amos, olayioni le Nahum, olayioni le Elsi, olayioni le Nagai, 26olayioni le Maat, olayioni le Matatias, olayioni le Semein, olayioni le Yosek, olayioni le Yuda, 27olayioni le Yonan, olayioni le Resa, olayioni le Serubabel, olayioni le Shealtiel, olayioni le Neri, 28olayioni le Melki, olayioni le Adi, olayioni le Kosam, olayioni le Elmadam, olayioni le Er, 29olayioni le Yoshua, olayioni le Eliesa, olayioni le Yorim, olayioni le Matar, olayioni le Lawi, 30olayioni le Simeon, olayioni le Yuda, olayioni le Yosef, olayioni le Yonam, olayioni le Eliakim, 31olayioni le Melea, olayioni le Mena, olayioni le Matata, olayioni le Natan, olayioni le Daudi, 32olayioni le Obed, olayioni le Boas, olayioni le Sala, olayioni le Nashon, 33olayioni le Aminadab, olayioni le Admin, olayioni le Arni, olayioni le Hesron, olayioni le Peres, olayioni le Yuda, 34olayioni le Yakobo, olayioni le Isaak, olayioni le Abraham, olayioni le Tera, olayioni le Nahor, olayioni le Serug, 35olayioni le Reu, olayioni le Peleg, olayioni le Eber, olayioni le Shela, 36olayioni le Kainan, olayioni le Alfasad, olayioni le Shem, olayioni le Noa, olayioni le Lamek, 37olayioni le Metusela, olayioni le Enok, olayioni le Yared, olayioni le Mahalalel, olayioni le Kainan, 38olayioni le Enos, olayioni le Seti, olayioni le Adam, olayioni le Nkai.
