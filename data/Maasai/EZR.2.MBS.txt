Esiana e lelo apa ootorrinyunye te yararoto
(Neh 7.4-73)
1Kulo apa lelo tung'ana leilo kerenket ooetuo te yararoto eing'uaa lelo apa ootarayie Nebukadnesa olaiguanani le Babilon arik eenata te Babilon. Etorrinyunye ninche aapuonu Yerusalem o Yuda nelo ake pooki tung'ani enkang enye makewan. 2Eetuo apa ninche tenebo Serubabel, o Yeshua, o Nehemia, o Seraia, o Reelaia, o Mordekai, o Balshan, o Mispar, o Bigfai, o Rehum, o Baana. Ena esiana oo ltung'ana loo Siraeli; 3ore ilayiok le Parosh era esiana enye nkalifuni are o iip nabo o ntomoni naapishana oare. 4Ore ilayiok le Shefatia, nera esiana enye, iip uni o ntomoni naapishana oare. 5Ore ilayiok le Ara, nera esiana enye iip naapishana o ntomoni naapishana o imiet. 6Ore ilayiok le Pahatmoab, aa taa ilayiok le Yeshua o Yoab, naa nkalifuni are o iip isiet o tomon oare. 7Ore ilayiok le Elam naa enkalifu nabo o iip are o onom o ong'uan. 8Ore ilayiok le Satu naa iip naaudo o artam o imiet. 9Ore ilayiok le Sakai, naa iip naapishana o ntomoni ile. 10Ore ilayiok le Bani naa iip ile o artam o are. 11Ore ilayiok le Bebai, naa iip ile o tikitam o okuni. 12Ore ilayiok le Asgad naa enkalifu nabo o iip are o tikitam oare. 13Ore ilayiok le Adonikam naa iip ile o ntomoni ile o ile. 14Ore ilayiok le Bigfai naa nkalifuni are o onom o ile. 15Ore ilayiok le Adin, naa iip ong'uan o onom o ong'uan. 16Ore ilayiok le Ater, aa taa ile Hesekia, naa ntomoni naaudo o isiet. 17Ore ilayiok le Besai naa iip uni o tikitam o okuni. 18Ore ilayiok le Yora naa, iip nabo o tomon oare. 19Ore ilayiok le Hashum naa, iip are o tikitam o okuni. 20Ore ilayiok le Gibar, naa ntomoni naaudo o imiet. 21Ore ilayiok le Betlehem naa iip nabo o tikitam o okuni. 22Ore iltung'ana le Netofia naa onom o ile. 23Ore iltung'ana le Anatot naa iip nabo o tikitam o isiet. 24Ore ilayiok le Asmafet naa artam oare. 25Ore ilayiok le Kiriataim, Kefira o Beerot naa iip naapishana o artam o okuni. 26Ore ilayiok le Rama o Geba, naa iip ile o tikitam o obo. 27Ore iltung'ana le Mikmas, naa iip nabo o tikitam oare. 28Ore iltung'ana le Betel o Ai naa iip are o tikitam o okuni. 29Ore ilayiok le Nebo, naa onom o are. 30Ore ilayiok le Makbish naa iip nabo o onom o ile. 31Ore ilayiok le likae le Elam naa enkalifu nabo o iip are o onom o ong'uan. 32Ore ilayiok le Harim, naa iip uni o tikitam. 33Ore ilayiok le Lod, o Hadid, o Ono, naa iip naapishana o tikitam o imiet. 34Ore ilayiok le Yeriko, naa iip uni o artam o imiet. 35Ore ilayiok le Seena naa nkalifuni uni o iip ile o tomoni uni.
36Ore ilapolosak lelo ayiok le Yedaia, le nkaji e Yeshua naa iip naaudo o ntomoni naapishana o okuni. 37Ore ilayiok le Imer naa, enkalifu nabo o onom oare. 38Ore ilayiok le Pashur, naa enkalifu nabo o iip are o artam oopishana. 39Ore ilayiok le Harim, naa enkalifu nabo o tomon oopishana.
40Ore Ilawi, lelo ayiok le Yeshua o Kadmiel, loo layiok le Hadafia, naa ntomoni naapishana o ong'uan. 41Ore ilaranyak: lelo ayiok le Asaf, naa iip nabo o tikitam o isiet. 42Ore lelo ayiok loo larripok loo nchomito; ilayiok le Shalum, o layiok le Ater, o layiok le Talmon o layiok le Akub, o layiok le Hatita, o layiok le Shobai, ore pooki tenebo, naa iip nabo o tomoni uni o oudo.
43Ore ilaisiaayiak le nkaji e Nkai, naa lelo aisiaayiak, o layiok le Siha, o layiok le Hafusa, o layiok le Taabaot, 44o layiok le Keros, o layiok le Siaha, o layiok le Padon, 45o layiok le Lebana, o layiok le Habaga, o layiok le Akub, 46o layiok le Hagab, o layiok le Shamlai, o layiok le Hanan, 47o layiok le Gidel, o layiok le Gahar, o layiok le Reaia, 48o layiok le Resin, o layiok le Nekoda, o layiok le Gasam, 49o layiok le Usa, o layiok le Pasea, o layiok le Besai, 50o layiok le Asna, o layiok le Meumin, o layiok le Nefisim, 51o layiok le Bakuk, o layiok le Hakufa, o layiok le Hahur, 52o layiok le Baslut, o layiok le Mehida, o layiok le Harsha, 53o layiok le Barkos, o layiok le Sisera, o layiok le Tema, 54o layiok le Nesia, o layiok le Hatifa.
55Ore too layiok loo sinkan le Solomon; naa ilayiok le Sotai, o layiok le Hosoferet, o layiok le Peruda, 56o layiok le Yaala, o layiok le Darkon, o layiok le Gidel, 57o layiok le Shefatia, o layiok le Hatil, o layiok le Pokeret-hasebaim, tenebo ilayiok le Ami.
58Ore ilaisiaayiak pooki le nkaji e Nkai o layiok loo sinkan le Solomon naa iip uni o ntomoni naaudo oare.
59Kulo apa lelo ooetuo eing'uaa Telmela, o Telhasha, o Kerub, o Adan, o Imer, hoo neitu apa ninche eidim aaibalie nkajijik oo loo menye anaa enetii ntipati enye, tenaa keitipat ninche Isirael: 60lelo ayiok le Delaia, o layiok le Tobia, o layiok le Nekoda, nera pooki iip ile o onom oare. 61Ore sii too layiok loo lapolosak, lelo ayiok le Barsilai, laa ninye apa oyama nabo oo ntoyie e Barsilai le Gilead, laa ninye apa eipotieki enkarna enye. 62Eing'orutua kulo enkigeroto enye tiatua lelo ooigeroki too iishoritin, kake eitu etumi ninche teine, neaku eiwuang'ieki ninche te mpolosare aajo mesinya; 63 etiaaka ninche olbalusui nchere pee meinos ina daa sinyati oleng, meteleku ake tenaa ketii olapolosani oikilikuanishore Urim o Tumim.
64Ore apa ina umata pooki tenebo naa nkalifuni artam o are o iip uni o ntomoni ile, 65metii isinkan lenye o nkaisiaayiak enye, laa ore tenebo ninche naa, nkalifuni naapishana o iip uni o tomoni uni oopishana, neeta apa ninche ilaranyak loo lewa o ngoruo iip are. 66Ore mbartan enye naa iip naapishana o tomoni uni o ile, ore inyumpui enye naa iip are o artam o imiet, 67ore ntamesin enye naa iip ong'uan o tomoni uni o imiet, ore isirkon lenye naa nkalifuni ile o iip naapishana o tikitam.
68Ore erubata oo lukuny oo lmareita, ina kata pee epuonu enkaji o Laitoriani te Yerusalem, neishoru nkishorot olpela te nkaraki enkaji e Nkai, pee esheti ninye te wueji enye; 69te risioroto e nkidimata enye eishorutua ninche aapik olchumati le siaai nkiloi iip imiet e sahabu, o nkiloi nkalifuni are o ntomoni isiet e nkosholai, o lkilani iip nabo loo lapolosak.
70 Ore ilapolosak o Lawi o embata oo ltung'ana nemany Yerusalem, o neemanita ninye, ore ilaranyak, o larripok loo nchomito, o laisiaayiak le nkaji e Nkai nebik too nkang'itie enye, ore Isiraeli pooki nebik too nkang'itie enye.
