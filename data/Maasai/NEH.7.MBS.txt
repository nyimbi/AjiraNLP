1Ore apa pee eidipayu esuntai naidipa nkikeneta oo nchomito, neidipaki aategelu ilarripok loo nchomito, o laranyak, o Lawi, 2naisho nanu, olalashe lai Hanani o Hanania meitasheiki Yerusalem. Amu oltung'ani apa oisiligayu neure Enkai alang ilkumok. 3Najoki nanu ninche, “Emincho ake nchomito e Yerusalem eboli ometaba anaa neilepu enkolong; naa ore eton eitashe aarripisho nchoo ninche eiken nepik isekenke ooikenieki nchomito. Entegelu ilarripok tiatua lelo oomanya Yerusalem, neitashe ake pooki ng'ae te rishata enye, neipirare ake oltung'ani enkaji enye.”
Esiana e lelo ootorrinyunye te yararoto
4Aisapuk apa ina anasa nelala, kake ore iltung'ana ootii atua naa kuti nemetii nkajijik naaidipaki aateshet.
5Nepik apa Enkai ndamunot aainei pee aiturrur ilkituaak, o larikok, o ltung'ana pee eiger pooki anaa ntipati oo lmareita, lelo apa ootangasa aapuonu, natum nanu eigeroki aaiko nji: 6Iltung'ana lo lkerenket kulo apa ooetuo eing'uaa eyararoto e lelo apa ootarayie Nebukadnesa olaiguanani le Babilon nenap aya, etorrinyunye ninche Yerusalem o Yuda nelo ake pooki ng'ae enkang enye. 7Eetuo apa ninche tenebo Serubabel, o Yoshua, o Nehemia, o Asaria, o Raamia o Nahamani, o Mordekai, o Bilshan o Misparet o Bigfai, o Nehum o Baana.
Ore esiana oo ltung'ana le Israel: 8lelo ayiok le Perosh, naa nkalifuni are o iip nabo o ntomoni naapishana oare. 9Ore ilayiok le Shefatia naa iip uni o ntomoni naapishana oare. 10Ore ilayiok le Ara, naa iip ile o onomm oare. 11Ore ilayiok le Pahatmoab, aa taa ilayiok le Yoshua o Yoab, naa nkalifuni are o iip isiet o tomon o isiet. 12Ore ilayiok le Elam naa enkalifu nabo o iip are o onom o ong'uan. 13Ore ilayiok le Setu naa iip isiet o artam o imiet. 14Ore ilayiok le Sakai, naa iip naapishana o ntomoni ile. 15Ore ilayiok le Binui, naa iip ile o artam o isiet. 16Ore ilayiok le Bebai naa iip ile o tikitam o isiet. 17Ore ilayiok le Asgad, naa nkalifuni are o iip uni o tikitam o are. 18Ore ilayiok le Adonikan naa iip ile ontomoni ile oopishana. 19Ore ilayiok le Bigfai naa nkalifuni are o ntomoni naapishana. 20Ore ilayiok le Adin naa iip ile o onomm o imiet. 21Ore ilayiok le Ater, a taa ile Hesekia naa ntomoni naaudo o isiet. 22Ore ilayiok le Hashum naa iip uni o tikitam o isiet. 23Ore ilayiok le Besai, naa iip uni o tikitam o ong'uan. 24Ore ilayiok le Harif naa iip o tomon oare. 25Ore ilayiok le Gibeon naa ntomoni naaudo o imiet. 26Ore iltung'ana le Betlehem o Netofa naa iip ontomoni isiet o isiet. 27Ore iltung'ana le Anatot naa iip o tikitam o isiet. 28Ore iltung'ana le Bet-asmapet naa artam oare. 29Ore iltung'ana le Kiriat-yearim, Kepira o Beerot, naa iip naapishana o artam o kuni. 30Ore iltung'ana le Rama o Geb, naa iip ile o tikitam o obo. 31Ore iltung'ana le Mikmas naa iip o tikitam o are. 32Ore iltung'ana le Betel o Ai, naa iip o tikitam okuni. 33Ore iltung'ana le nkae Nebo naa onomm o are. 34Ore ilayiok le likae oji Elam naa enkalifu nabo o iip are o onomm o ong'uan. 35Ore ilayiok le Harim, naa iip uni o tikitam. 36Ore ilayiok le Yeriko naa iip uni o artam o imiet. 37Ore ilayiok le Lod, Hadid o Ono, naa iip naapishana o tikitam o obo. 38Ore ilayiok le Senaa naa nkalifuni naapishana o iip naaudo o tomoni uni.
39Ore ilapolosak: lelo ayiok le Yedaia aa taa enkaji e Yoshua, naa iip naaudo o ntomoni naapishana o okuni. 40Ore ilayiok le Imer, naa enkalifu nabo o onomm o are. 41Ore ilayiok le Pashur, naa enkalifu nabo o iip are o artam o oopishana. 42Ore ilayiok le Harim naa enkalifu nabo o tomon oopishana.
43Ore Ilawi: lelo ayiok le Yoshua, aa taa ile Kadmiel loo layiok le Hedefa, naa ntomoni naapishana o ong'uan. 44Ore ilaranyak lelo ayiok le Asaf, naa iip nabo o artam o isiet. 45Ore ilarripok loo nchomito: lelo ayiok le Shalum, ilayiok le Ater, lelo ayiok le Talmon, lelo ayiok le Akub, lelo ayiok le Hatita, lelo ayiok le Shobai, naa iip nabo o tomoni uni o isiet.
46Ore lelo aisiaayiak le nkaji e Nkai, lelo ayiok le Siha, lelo ayiok le Hasufa, lelo ayiok le Taabaot, 47lelo ayiok le Keros, lelo ayiok le Sia, lelo ayiok le Padon 48lelo ayiok le Lebana, lelo ayiok le Hagaba lelo ayiok le Shalmai, 49lelo ayiok le Hanan, lelo ayiok le Gidel, lelo ayiok le Gahar, 50lelo ayiok le Reia, lelo ayiok le Resin, lelo ayiok le Nekoda, 51lelo ayiok le Gesam, lelo ayiok le Usa, lelo ayiok le Pasea, 52lelo ayiok le Besai, lelo ayiok le Meunim, lelo ayiok le Nefushesim, 53lelo ayiok le Bakbuk, lelo ayiok le Hakufa, lelo ayiok le Hashur, 54lelo ayiok le Bashit, lelo ayiok le Mehida, lelo ayiok le Harsha, 55lelo ayiok le Barkos lelo ayiok le Sisera, lelo ayiok le Tema, 56lelo ayiok le Nesia, lelo ayiok le Hatifa.
57Ore ilayiok loo sinkan le Solomon, ilayiok le Sotai, ilayiok le Soferet, ilayiok le Persida, 58ilayiok le Yaala, ilayiok le Darkon, ilayiok le Gidel, 59ilayiok le Shefatia, ilayiok le, Hatil, ilayiok le Pokeret-Lasebaim, lelo ayiok le Amon. 60Ore isinkan pooki le nkaji e Nkai, o sinkan loo layiok le Solomon pooki naa iip uni o ntomoni naaudo oare.
61Kulo oodukino apa lelo ooetuo eing'uaa Telmela, Telhasha, Keurub, Adon, o Imer, kake meidim apa ninche aaibalie ilmareita loo loo menye aashu ineeikino ninche, tenaa ke ltung'ana le Israel ninche. 62Ilayiok le Delaia, ilayiok le Tobia, ilayiok le Nekoda, naa iip ile o artam oare. 63Ore sii ilapolosak: lelo ayiok le Hobaia, lelo ayiok le Hakos, ilayiok le Barsilai, laa ninye apa oyama entito nabo too ntoyie e Barsilai le Gilead neaku ninche eipotieki). 64Eing'orutua kulo enkigeroto oo nkarn enye tenebo lelo ooigeroki too lmareita, kake eitu etumi teine, neaku etundung'oyioki too lapolosak aajo keiduruk; 65 etiaaka olbalusui ninche pee mejing ninche enkinosata eina daa sinyati aisul, ometaba anaa nedumunye olapolosani oata Urim o Tummim.
66Ore apa eumata pooki tenebo naa nkalifuni artam oare o iip uni o ntomoni ile 67metii isinkan lenye lewa o sinkan enye ng'oruo, nera apa ninche nkalifuni naapishana o iip uni o ntomoni ile o opishana; neeta apa ninche ilaranyak iip are o artam o imiet, ilewa o ng'oruo. 68Ore mbartan enye naa iip naapishana o tomoni uni o ile. Ore inyumpui enye naa iip are o artam o imiet. 69Ore ntamesin enye naa iip ong'uan o tomoni uni o imiet. Ore isirkon lenye naa nkalifuni ile o iip naapishana o tikitam.
70Ore apa ilukuny oo nkajijik oo loo menye neishooyo nkishorot tenkaraki esiaai. Eishooyie olbalusui apik olchumati nkiloi isiet e sahabu, o ngirran onomm, o lkilani loo lapolosak iip imiet o tomoni uni. 71Ore ilkulikae oora ilukuny oo nkajijik oo loo menye neishooyo aapik olchumati le siaai sahabu nkiloi iip o ntomoni ile, o imiet, o enkosholai nkiloi enkalifu nabo o iip are o onomm. 72Ore ilkulikae tung'ana neishooyo sahabu nkiloi iip nabo o ntomoni ile o isiet, o nkiloi iip nabo o artam e nkosholai, o lkilani ntomoni ile oopishana loo lapolosak.
73 Neaku ore ilapolosak, lelo le Lawi, o larripok loo nchomito, o laranyak, o embata oo ltung'ana, o embata oo sinkan le nkaji e Nkai, o Siraeli pooki; netoni too nkang'itie enye.
Ore pee ebau olapa liopishana, netii nkera e Israel nkang'itie enye maate.
